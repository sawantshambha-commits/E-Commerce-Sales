# 📊 E-Commerce Sales Analysis & Dashboard

This project analyzes large-scale e-commerce data (10K+ records) to generate actionable business insights.  
The dataset simulates real-world scenarios similar to platforms like Amazon and Flipkart.

The project includes:
- SQL-based data generation and analysis
- Data modeling using star schema
- Interactive Power BI dashboard

---

## 🎯 Business Objectives
- Analyze revenue trends and growth
- Identify top customers and products
- Understand customer behavior
- Evaluate payment success and failures
- Track order performance

---

## 🛠 Tools & Technologies
- MySQL
- Power BI
- DAX
- SQL

---

## 📊 Key Insights
- Monthly revenue trends 📈
- Top-performing products 🛒
- High-value customers 👤
- Payment success rate 💳
- Order status distribution 📦
